VisionSentinel  is a computer vision based surveillance analysis system designed to automatically understand visual scenes and detect potential security risks from images or video footage. The system takes an image or CCTV video as input and processes it using a YOLOv8 object detection model to identify objects such as people, bags, vehicles, and other items present in the scene. After detecting objects, a rule-based risk evaluation module analyzes the detected objects and their relationships to identify suspicious situations such as unattended bags, crowd gatherings, or potentially dangerous objects. The system then generates a structured report describing the scene, detected objects, and the calculated risk level. The backend is built using FastAPI to handle API requests and processing logic, while a simple Streamlit interface is used to allow users to upload images or videos and view the analysis results. OpenCV is used for video frame processing and visualization, and PostgreSQL is used to store analysis results and reports. This project demonstrates how computer vision and AI can be combined to assist in automated surveillance monitoring and risk detection for security applications.
